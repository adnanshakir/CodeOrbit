# CodeOrbit

CodeOrbit is an AI-powered cloud development platform that enables users to build, edit, test, and run applications in isolated Kubernetes sandboxes. Instead of setting up a local development environment, users can create a secure workspace in the cloud, interact with an integrated AI assistant to generate or modify code, manage project files through a built-in file explorer, use a live terminal, and instantly preview their applications in real time.

The platform is designed around a microservices architecture, with each sandbox provisioned on demand to provide an isolated and scalable development environment. This approach allows multiple users to work independently while ensuring security, resource isolation, and efficient container management.

CodeOrbit is being developed using React, Node.js, Express, Kubernetes, Docker, MongoDB, Redis, RabbitMQ, Socket.IO, and Google OAuth to deliver a modern cloud-based development experience focused on scalability, real-time interaction, and secure code execution.

> **Note:** CodeOrbit is currently under active development and is not yet deployed. Features, architecture, and documentation will continue to evolve as the project progresses.
